# Job Prediction System

## Project Overview

This project analyzes recruitment and placement data using Python and Streamlit. The objective is to identify the key factors influencing candidate placement outcomes through data preprocessing, exploratory data analysis (EDA), machine learning model development, and an interactive analytics dashboard.

## Key Insights

* Technical assessment scores significantly influence placement outcomes.
* Higher interview performance is associated with greater placement success.
* Candidates with relevant certifications generally achieve better placement rates.
* Previous work experience positively impacts placement opportunities.
* Company tier and job role alignment contribute to recruitment success.
* The Random Forest Classifier achieved the best overall classification performance.
